Para executar o programa, faça:
>>> python3 TP1.py ALG_ID INIT_STATE PRINT

Onde ALG_ID é o identificador do algoritmo, i.e., um valor em ['A','B','I','G','H','U']
INIT_STATE é uma sequência de nove valores indicando as posições das peças no tabuleiro
PRINT é um parâmetro opcional para imprimir o caminho encontrado
